2. 动态规划

经过观察发现如下：

  • n==0, result = [‘’]
  • n==1, result = [
    () = ( + result_0 + ) + result_0
    ]
  • n==2, result = [
    ()() = ( + result_0 + ) + result_1()
    (()) = ( + result_1() + ) + result_0
    ]
  • n==3, result = [
    ()()() = ( + result_0 + ) + result_2_1()()
    ()(()) = ( + result_0 + ) + result_2_2(())
    (())() = ( + result_1() + ) + result_1()
    (()()) = ( + result_2_1()() + ) + result_0
    ((())) = ( + result_2_2(()) + ) + result_0
    ]
    因此我们可以得出如下结论：
  • dp[n] = [( + x + ) + y for x in dp[j] for y in dp[i - j - 1]] , j in range(i)。
